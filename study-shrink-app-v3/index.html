PDF text extraction, local file import, cleaner, chunker, prompt builder.

PDF extraction added

This version uses PDF.js to extract selectable text from PDFs. It works best for normal digital PDFs. It does not OCR scanned/image-only PDFs, and complex multi-column tables may need manual cleanup.
