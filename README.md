## AI Edge Gallery setup

1. Open Google AI Edge Gallery on iPhone.
2. Go to Agent Skills.
3. Tap Skills.
4. Tap +.
5. Choose Load from URL.
6. Paste the folder URL:

https://DPropst-Creatine.github.io/ai-edge-skills/cruise-food-analyzer-js/

Do not paste the direct SKILL.md URL into the app.

## First test prompts

Set profile:

```text
Use cruise-food-analyzer.

Set my profile:
Age: 40
Sex: male
Height: 69 inches
Weight: 185 lb
Goal: vacation damage-control
Activity: moderate
Training days: 3
```

Check profile:

```text
Use cruise-food-analyzer.

What are my current cruise food goals and targets?
```

Log a meal:

```text
Use cruise-food-analyzer.

Analyze and log this meal: chicken with fried rice, peas, carrots, and sauce or oil.
Estimate satiety, calories/macros, glycemic impact, and how it fits today.
```
